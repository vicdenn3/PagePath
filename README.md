# 📖 PagePath
**Интерактивное исследование книг — карты, персонажи, связи, сообщество**
## О проекте

PagePath превращает чтение из пассивного процесса в **интерактивное исследование**. Платформа помогает читателям ориентироваться в сложных произведениях: запоминать персонажей, отслеживать локации на реальной карте и видеть, как связаны герои между собой.

Особенно актуально для объёмных книг с разветвлённым сюжетом и большим количеством действующих лиц. PagePath объединяет современные инструменты визуализации с читательским сообществом — всё в одном месте.

> Чтение + визуализация + общение = новый уровень погружения в книгу.
## Функциональность

###  Каталог локаций
Все места из книги отображены на реальной карте. К каждой локации прикреплены текстовые описания, цитаты из текста и фотоматериалы — архивные снимки, современные фото или AI-генерация.

###  Медиатека персонажей
Галерея героев с иллюстрациями, кадрами из экранизаций и кратким описанием каждого персонажа. Удобно, когда в книге несколько десятков действующих лиц.

###  Карта связей
Визуальный граф взаимоотношений между персонажами: родство, дружба, конфликты, союзы. Позволяет видеть всю сеть отношений одним взглядом.

###  Форум
Площадка для обсуждения книги: теории, разбор сцен, сравнение экранизаций. Вместо разрозненных чатов и соцсетей — одно структурированное место для общения фанатов.

###  Энциклопедия
Дополнительные материалы: интересные факты, история создания произведения, рецензии, оригинальный текст. Всё, что делает чтение глубже.
